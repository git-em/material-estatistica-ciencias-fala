Teste de coerência entre juízes, Kappa de Fleiss, alfa de Krippendorff

1. Para esse teste é preciso instalar o pacote "irr";
2. Para tanto, no R, vá no Menu Packages and Data (ou "pacotes e dados", a depender  da língua), selecione  Package installer, escreva na busca irr e então Get list. 
Quando aparecer o irr, selecione o pacote na lista e, abaixo, clique no botão de instalação;
3.Uma vez instalado, no prompt do R, escreva: library(irr);
Pronto, o pacote está carregado.

Para o teste de coerência de Kappa pelo método de Fleiss ou o alfa de Krippendorff, leia primeiramente a tabela com a funcão read.table(), sem o header, porque a tabela não precisa ter indicação de variável:

data < read.table(file.choose())

A tabela deve ser uma matriz em que cada linha representa um objeto avaliado e cada coluna um juiz. Por exemplo, na seguinte, com 3 juízes e 5 objetos avaliados e 3 categorias A, B e C:

A A B 
B C A
A A A
B B C
A B B

A matriz pode ser formada de números ou letras, mas a função que faz o teste de coerência tratará os valores como categorias.

4. Para terminar, o teste referido acima é feito com:

kappam.fleiss(data)  # para valores nominais

kripp.alpha(datamatrixtransp, method="interval") # apra valores intervalares

Pronto, pode examinar os resultados, anotando:

o valor de kappa/alfa (coeréncia de 0 a 1)
o valor de z 
o valor de p (se menor que o nível de significãncia, rejeita-se a hipótese nula de que o kappa é zero).
